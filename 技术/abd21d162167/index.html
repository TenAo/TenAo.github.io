  3. footer内增加本站运行天数。

    代码
    <% if (theme.footer.operate_time) { %>
  <div>
    <span id="timeDate">载入天数...</span><span id="times">载入时分秒...</span>
    <script>
    var now = new Date();
    function createtime() {
      var grt= new Date("01/01/2021 00:00:00");
      now.setTime(now.getTime()+250);
      days = (now - grt ) / 1000 / 60 / 60 / 24; dnum = Math.floor(days);
      hours = (now - grt ) / 1000 / 60 / 60 - (24 * dnum); hnum = Math.floor(hours);
      if(String(hnum).length ==1 ){hnum = "0" + hnum;} minutes = (now - grt ) / 1000 /60 - (24 * 60 * dnum) - (60 * hnum);
      mnum = Math.floor(minutes); if(String(mnum).length ==1 ){mnum = "0" + mnum;}
      seconds = (now - grt ) / 1000 - (24 * 60 * 60 * dnum) - (60 * 60 * hnum) - (60 * mnum);
      snum = Math.round(seconds); if(String(snum).length ==1 ){snum = "0" + snum;}
      document.getElementById("timeDate").innerHTML = "本站已在宇宙中运行 "+dnum+" 天 ";
      document.getElementById("times").innerHTML = hnum + " 小时 " + mnum + " 分 " + snum + " 秒啦";
    }
    setInterval("createtime()",250);
    </script>
  </div>
<% } %>

  3. 文章内链接颜色优化。

    ..\themes\cards\source\css\style\_base\defines.styl

    ..\themes\cards\source\css\style\_pages\post.styl

    代码内容

    defines.styl文件内添加定义：

    $color_post_link = convert(hexo-config('style.color.post_link_color')) || #a5dff9

    post.styl 文件内.post__content和.post-entry__content下增加样式：

    /*自行添加*/
a
    color $color_post_link
/*添加完毕*/

    post content就是文章界面文章内容的link颜色。post-entry content就是首页的文章内容里的link颜色，即下图的蓝色链接。

    不过这个其实不太用得到，正常人谁在description里放链接啊（ry

    最后再在主题配置文件内style->color下新添一个值post_link_color，就可以比较方便地更改颜色了。

    多说一句，网页设计只要不超过4种颜色，基本上不会丑到哪里去。

  4. 评论系统使用Gitalk，设置参考Cards官方文档。申请Application的时候设置了域名就填写完整域名，不要填原本的仓库link（xxx.github.io），否则无法正常使用。
